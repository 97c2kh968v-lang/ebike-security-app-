# 深远GT1防盗控制器 — 嘉立创PCB下单操作指引

## 第一步：安装工具

### 方案A：立创EDA标准版（推荐，最简单）
1. 访问 `https://lceda.cn/` 下载 **立创EDA标准版**（免费）
2. 安装后注册登录，用嘉立创账号（没有就注册一个）
3. 支持Windows / macOS / Linux

### 方案B：KiCad（专业，功能强）
1. 访问 `https://www.kicad.org/` 下载 **KiCad 8.0+**
2. 安装后导入嘉立创元件库：`https://github.com/richardhmmming/嘉立创庫`

---

## 第二步：画原理图（两种方法）

### 方法A：立创EDA（推荐）

1. 打开立创EDA → 新建工程 → 命名 "ebike_security"
2. 点击 **"图纸"** 开始画原理图
3. 放置元件（按BOM表的型号搜索）：
   - 搜索 `ESP32-WROVER-E` → 放置
   - 搜索 `MP2307` → 放置
   - 搜索 `LM2596S-5.0` → 放置
   - 搜索 `AMS1117-3.3` → 放置
   - 搜索 `SRD-12VDC` → 放置2个（门锁+座桶）
   - 搜索 `PC817C` → 放置4个
   - 搜索 `MAX98357A` → 放置
   - 搜索 `74HC14D` → 放置
   - 搜索 `SMBJ58A` → 放置
   - 搜索 `SS54` → 放置
   - 搜索 `XT60` → 放置
   - 搜索 `TerminalBlock_2P` → 放置3个
   - 搜索 `LED-3mm` → 放置2个
4. 连线：
   - 电源路径：XT60 → SMBJ58A → F1 → MP2307(VIN+) → LM2596S(VIN+) → AMS1117(VIN) → ESP32(3V3)
   - 地线：所有GND连一起
   - 继电器控制：ESP32 GPIO4 → PC817(1) → PC817(3) → S8050 Base → 继电器线圈
   - 震动报警：报警信号 → PC817(1) → PC817(4) → ESP32 GPIO12
   - I2S音频：ESP32 GPIO18/GPIO19/GPIO21 → MAX98357A BCK/WS/DIN
5. 标注编号：点击 **"标注"** → 全部标注
6. 保存

### 方法B：用现有文件导入

1. 打开立创EDA → 新建工程
2. 导入CSV BOM：顶部菜单 **"制造"** → **"BOM表"** → **"导入BOM"**
3. 选择 `BOM嘉立创.csv` 文件
4. 系统会自动匹配立创元件库，检查确认

---

## 第三步：PCB布局（画板子）

1. 原理图完成后，点击 **"设计"** → **"转换到PCB"**
2. 设置板子尺寸：
   - 板子形状：矩形
   - 宽度：80mm
   - 高度：60mm
   - 层数：2层板
   - 板厚：1.6mm
3. 摆放元件（参考布局建议）：
   - 左边：电源输入（XT60、保险丝、TVS）
   - 中左：DC-DC降压（MP2307、LM2596）
   - 中间：ESP32-WROVER-E（主控）
   - 中右：继电器（门锁、座桶）
   - 右边：音频（MAX98357A、喇叭接口）
   - 底部：接插件（报警输入、座桶输出、扩展口）
4. 布线：
   - 电源线：尽量短粗（≥1.5mm）
   - 信号线：常规（0.3mm）
   - 地线：铺铜（顶层+底层都铺）
5. 设计规则检查（DRC）：
   - 线宽：最小0.2mm
   - 间距：最小0.2mm
   - 过孔：外径0.6mm，内径0.3mm
6. 保存

## 第四步：下单前的检查

### 必查项目
- [ ] 电源电压：48V系统用SMBJ58A，60V系统用SMBJ70A
- [ ] 继电器触点容量：≥10A
- [ ] 所有接插件都有对应的 footprint
- [ ] GND和电源线够粗
- [ ] 光耦方向正确（输入侧和输出侧不接反）
- [ ] ESP32的3.3V电源稳定（加足够去耦电容）

### 导出Gerber文件
1. 立创EDA：**"制造"** → **"PCB制板文件"**
2. 勾选所有层（顶层、底层、丝印、阻焊、钻孔）
3. 点击 **"生成制造文件"**
4. 下载ZIP压缩包

## 第五步：嘉立创下单（SMT贴片可选）

### 只做裸板（最便宜）
1. 访问 `https://www.jlc.com/` 登录账号
2. 点击 **"PCB下单"**
3. 上传Gerber文件（ZIP包）
4. 参数设置：
   - 板子尺寸：80×60mm
   - 层数：2层
   - 板厚：1.6mm
   - 铜厚：1oz
   - 阻焊颜色：绿色（便宜）/ 黑色（+20元）
   - 丝印颜色：白色
   - 数量：5片（打样价¥50）
   - 交货周期：24小时（加急）/ 3-4天（普通）
5. 点击 **"提交订单"** → 付款
6. 等待收货（一般3-5天）

### SMT贴片（帮你焊元件，省事）
1. 下单时在 **"SMT贴片"** 处选择 **"是"**
2. 上传BOM表（`BOM嘉立创.csv`）
3. 上传坐标文件（立创EDA可直接导出）
4. 系统自动匹配元件，确认BOM准确率
5. 确认后提交订单
6. 价格：SMT服务费 + 元件费（约¥200-300，5片）
7. 收货后直接可用，不用自己焊接

## 第六步：焊接与调试

### 自己焊接（裸板）
1. 买齐BOM表所有元件
2. 焊接顺序：
   - 先焊电源部分（DC-DC、LDO）
   - 再焊ESP32（建议用热风枪，小心别吹跑）
   - 最后焊继电器、接插件
3. 上电前必查：
   - 电源输入极性（别接反！）
   - 3.3V输出正常（用万用表测）
   - 没有短路（测输入阻抗）

### 调试步骤
1. 上电，测3.3V、5V、12V是否正常
2. 用USB-TTL接ESP32串口，烧录固件（Arduino IDE）
3. 手机打开nRF Connect，搜索 "深远GT1"，能搜到说明蓝牙正常
4. 测试继电器动作（APP点解锁/上锁）
5. 模拟震动报警（短接报警输入到GND），应该响音效

---

## 第七步：安装到车上

1. 断电操作！先拆掉电动车电池连接
2. 找到原车电门锁线（一般红/黑/绿三根）
3. 把控制器串联在电门锁线上（继电器常开触点）
4. 接报警器信号线（原车报警器有触发输出，接GPIO12）
5. 接座桶锁线（脉冲触发，接继电器常开触点）
6. 喇叭装在座桶内（避免淋雨）
7. 所有接线用热缩管绝缘
8. 控制器装在防水盒里，固定好
9. 恢复供电，测试所有功能

## 常见问题

### Q：嘉立创下单后多久能收到？
A：裸板3-5天，SMT贴片5-7天（加急24小时出货，加¥50）

### Q：BOM表里的LCSC编号是什么？
A：立创商城元件编号，下单SMT时系统自动识别，直接采购元件

### Q：ESP32怎么烧固件？
A：用USB-TTL线，TX/RX接ESP32的RX/TX，GPIO0接GND（进入下载模式），用Arduino IDE上传

### Q：蓝牙搜不到设备？
A：检查ESP32的3.3V供电是否正常，天线是否连接，代码里BLE初始化是否成功

### Q：继电器不动作？
A：用万用表测光耦输出侧电压，确认PC817导通时继电器线圈有12V
