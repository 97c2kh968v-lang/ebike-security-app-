# 深远GT1防盗控制器 - 嘉立创EDA 10分钟快速绘图指南

## 概述
本指南帮助你在**嘉立创EDA（免费网页版）**中，10分钟内完成原理图绘制，并直接下单生产PCB。

---

## 第一步：打开嘉立创EDA

1. 浏览器访问：https://lceda.cn
2. 点击右上角**"免费使用"**
3. 微信扫码 / 手机号注册登录
4. 点击**"标准版"** → **"新建工程"**
   - 工程名称：`深远GT1防盗控制器`
   - 原理图名称：`main`

---

## 第二步：批量导入BOM（自动放置元件）

1. 在嘉立创EDA顶部菜单 → **"设计"** → **"导入BOM"**
2. 下载本项目提供的 **`BOM嘉立创.csv`**（已含LCSC编号）
3. 点击**"选择文件"** → 上传 `BOM嘉立创.csv`
4. 系统自动匹配立创商城零件，确认后点击**"开始导入"**
5. ✅ 所有元件自动放置到原理图上！

> **提示**：如果某些零件未匹配，手动搜索LCSC编号添加（BOM.csv中已提供编号）

---

## 第四步：检查与标注

1. 点击顶部**"设计管理器"**（左侧面板）
2. 检查：
   - ✅ 所有元件都有**唯一的位号**（R1, R2, C1, U1...）
   - ✅ 所有网络都正确连接（无悬浮引脚）
   - ✅ 电源和GND网络正确标注
3. 点击**"工具"** → **"检查规则错误"**
4. 如有错误，按提示修复

---

## 第五步：转为PCB并下单

1. 顶部菜单 → **"设计"** → **"转为PCB"**
2. PCB布局建议：
   - 尺寸：80mm × 60mm（可放入电动车控制器盒）
   - 层数：双面板（足够用）
   - 板厚：1.6mm
   - 铜厚：1oz（常规）
3. 布局完成后，点击**"下单"** → 直接跳转**嘉立创PCB下单页面**
4. 上传Gerber文件（嘉立创EDA自动生成）
5. 选择：**SMT贴片**（自动焊接所有元件，省事！）
6. 支付，等待收货（通常3-5天）

## 常见问题

**Q: 找不到某个元件怎么办？**
A: 在嘉立创EDA左侧"元件库"搜索功能，输入型号（如"PC817"），从立创商城选择带有**"SMT"**标识的型号。

**Q: 连线很乱怎么办？**
A: 使用"网络标签"（Net Label）代替长距离连线。比如在所有需要3.3V的地方放一个"3.3V"网络标签。

**Q: PCB布局不知道怎么摆放？**
A: 嘉立创EDA有**"自动布局"**功能（工具 → 自动布局），或参考SVG原理图（`ebike_security_schematic.svg`）的大致布局。

**Q: SMT贴片贵吗？**
A: 嘉立创SMT：首5片免费（需支付钢网费约50元），之后2-5元/点。本项目的BOM约30个元件，预计总成本**80-120元**（含PCB+SMT+元件）。
